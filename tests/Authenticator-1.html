Mobile Authenticator Setup

1. Install an authenticator application on your phone. The applications listed here are supported.

  • FreeOTP
  • Google Authenticator

2. Open the application and scan the barcode.

3. Enter the one-time code provided by the application and click Save to finish the setup.

Cannot Scan?

2. Open the application, tap add an account, then select manual entry.

3. Enter your email address and this key.

Want to scan the barcode?

4. Enter the one-time code provided by the application and click Finish to complete the setup.
